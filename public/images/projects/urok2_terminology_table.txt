=============================================================================
ГЛАВНАЯ ФОРМУЛА: РЦК-ФОТ
=============================================================================

СТРУКТУРА:
---------------------------------------------------------------------------
РЦК – ОСНОВА (обязательно)
---------------------------------------------------------------------------
| Р | Роль        | Кто ты для ИИ?                                    |
| Ц | Цель        | Что делаем?                                       |
| К | Контекст    | Какие данные?                                     |
---------------------------------------------------------------------------
ФОТ – ОФОРМЛЕНИЕ (желательно)
---------------------------------------------------------------------------
| Ф | Формат      | Как оформляем?                                    |
| О | Ограничения | Чего избегать?                                    |
| Т | Тон         | Как звучим?                                       |
---------------------------------------------------------------------------


=============================================================================
ДЕТАЛЬНОЕ ОПИСАНИЕ ЭЛЕМЕНТОВ ФОРМУЛЫ
=============================================================================

[Р] РОЛЬ
---------------------------------------------------------------------------
ОПРЕДЕЛЕНИЕ:
Первое, что нужно указать. Это как представиться: "Здравствуйте, 
я начальник отдела кадров".

ЧТО ДАЁТ:
- подбирает соответствующую лексику
- использует правильные термины
- выдерживает нужный уровень формальности
- учитывает профессиональную специфику

БАЗОВЫЕ РОЛИ:
- Ты – опытный делопроизводитель в государственном учреждении
- Ты – юрист администрации города со стажем 10 лет
- Действуй как пресс-секретарь мэрии
- Ты – специалист отдела кадров, знающий ТК РФ

КОМБИНИРОВАННЫЕ РОЛИ (мощнее):
- Ты – начальник отдела закупок с 15-летним опытом, 
  специализирующийся на 44-ФЗ
- Ты – помощник руководителя, который отлично знает деловой 
  этикет и умеет дипломатично формулировать отказы


[Ц] ЦЕЛЬ
---------------------------------------------------------------------------
ОПРЕДЕЛЕНИЕ:
После роли сразу указываем, что нужно сделать.

ГЛАГОЛЫ-КОМАНДЫ ДЛЯ ГОССЛУЖБЫ:
- составь – для официальных документов
- сформулируй – для точных формулировок
- подготовь – для черновиков
- переформулируй – для улучшения текста
- сократи – для уменьшения объёма
- расширь – для добавления деталей
- проверь – для поиска ошибок
- преобразуй – для смены формата


[К] КОНТЕКСТ
---------------------------------------------------------------------------
ОПРЕДЕЛЕНИЕ:
Вся нужная информация для выполнения задачи.

СТРУКТУРА КОНТЕКСТА:
1. От кого и кому
   От: [должность, организация]
   Кому: [адресат]

2. Суть дела
   По вопросу: [тема]
   Проблема: [описание]

3. Ключевые факты и цифры
   Дата: [конкретная дата]
   Количество: [число]
   Бюджет: [сумма]


[Ф] ФОРМАТ
---------------------------------------------------------------------------
ОФИЦИАЛЬНЫЕ ДОКУМЕНТЫ:
- служебная записка
- объяснительная записка
- письмо-ответ на обращение
- протокол совещания
- справка

РАБОЧИЕ МАТЕРИАЛЫ:
- тезисы для выступления (буллет-поинты)
- презентация (заголовок + 3-4 пункта на слайд)
- таблица сравнения
- чек-лист проверки
- план мероприятия


[О] ОГРАНИЧЕНИЯ
---------------------------------------------------------------------------
ТИПИЧНЫЕ ОГРАНИЧЕНИЯ:
- без сложных юридических терминов
- не используй сокращений, кроме общепринятых
- без обещаний конкретных сроков
- не упоминай внутренние проблемы организации
- объём – не более 1 страницы А4


[Т] ТОН
---------------------------------------------------------------------------
ШПАРГАЛКА ПО ТОНАМ:

Для вышестоящих органов:
→ официальный, уважительный, без фамильярности

Для коллег того же уровня:
→ деловой, но дружелюбный

Для подчинённых:
→ чёткий, мотивирующий, без приказного тона

Для граждан:
→ вежливый, понятный, без канцелярита

Универсальный деловой:
→ профессиональный, нейтральный, корректный


=============================================================================
ШПАРГАЛКА: КАКУЮ РОЛЬ ВЫБРАТЬ?
=============================================================================

ДЛЯ ДОКУМЕНТОВ → профильный специалист
---------------------------------------------
протокол                → секретарь
юридический ответ       → юрист
техническое задание     → специалист по закупкам

ДЛЯ ПИСЕМ → учитывайте адресата
---------------------------------------------
письмо наверх          → помощник руководителя
письмо коллегам        → специалист того же уровня
письмо гражданам       → специалист по работе с обращениями

ДЛЯ АНАЛИТИКИ → эксперт
---------------------------------------------
анализ данных          → аналитик
сравнение вариантов    → консультант
оценка рисков          → риск-менеджер

ЛАЙФХАКИ ОТ БЫВАЛЫХ
=============================================================================

ЛАЙФХАК 1: "Пример для подражания"
→ Напиши в стиле этого письма: [вставляете удачное письмо]

ЛАЙФХАК 2: "Пошаговая инструкция"
→ Шаг 1: Напиши введение с благодарностью за обращение
  Шаг 2: Изложи нашу позицию со ссылкой на регламент
  Шаг 3: Предложи альтернативное решение
  Шаг 4: Завершающая фраза с контактами для связи

ЛАЙФХАК 3: "Антипримеры"
→ НЕ пиши так: 'Ваше обращение рассмотрено, меры приняты'

ЛАЙФХАК 4: "Человечность"
→ Пиши простым человеческим языком
→ Избегай штампов и канцелярита
→ Как если бы объяснял коллеге за чаем


=============================================================================
КАК "ОЧЕЛОВЕЧИТЬ" ТЕКСТ ОТ ИИ
=============================================================================

ПРИЗНАК 1: Длинные тире (–)
ИСПРАВЛЕНИЕ: Заменяем на обычные (-) или двойные (--)

ПРИЗНАК 2: Излишняя структурированность
ИСПРАВЛЕНИЕ: Убираем лишние подзаголовки
             Объединяем короткие абзацы

ПРИЗНАК 3: Повторяющиеся вводные
ИСПРАВЛЕНИЕ: "Важно отметить", "Следует подчеркнуть", "Кроме того"
             → Оставляем 1-2, остальные убираем или заменяем

ПРИЗНАК 4: Идеальная грамматика там, где не надо
ИСПРАВЛЕНИЕ: В дружеских письмах можно начать с "И" или "Но"
             Можно использовать восклицания (в меру!)

ПРИЗНАК 5: Обтекаемые формулировки
ИСПРАВЛЕНИЕ: "Рассмотрим в ближайшее время" → "Рассмотрим до 20 декабря"
             "Приложим усилия" → "Сделаем"

ПРИЗНАК 6: Большая буква после двоеточия
ИСПРАВЛЕНИЕ: "Мероприятие: Уборка территории" → "Мероприятие: уборка территории"
             Это нарушение орфографии и 100% признак ИИ-текста

ЧАСТЫЕ ОШИБКИ НАЧИНАЮЩИХ
=============================================================================

ОШИБКА 1: "Забыл роль"
✖ Плохо:  Напиши ответ о проделанной работе
✓ Хорошо: Ты – начальник IT-отдела. Напиши квартальный отчёт

ОШИБКА 2: "Телепатия"
✖ Плохо:  Допиши письмо
✓ Хорошо: Допиши письмо, добавив извинения за задержку и новый срок

ОШИБКА 3: "Война и мир"
✖ Плохо:  Промпт на 2 страницы с массой деталей
✓ Хорошо: Чёткий промпт с ключевой информацией

ОШИБКА 4: "Сделай красиво"
✖ Плохо:  Напиши покрасивее
✓ Хорошо: Используй эпитеты, добавь благодарность, тёплое пожелание

ОШИБКА 5: "Без проверки"
✖ Плохо:  Получили текст от ИИ → сразу отправили
✓ Хорошо: Всегда проверяем факты, даты, имена, должности
