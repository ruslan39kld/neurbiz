РАЗДЕЛ 2: Универсальный промпт-генератор формул
-----------------------------------------------------------------------------
[ПРАКТИЧЕСКИЙ КЕЙС] Шаблон для DeepSeek/ChatGPT/Claude:

Ты – эксперт по Microsoft Excel с 20-летним стажем. 
Напиши формулу для следующей задачи:

ЧТО СЧИТАЕМ: [описание задачи]

ГДЕ ДАННЫЕ: [указать ячейки/столбцы]

УСЛОВИЯ: [если есть]

РЕЗУЛЬТАТ: [что хотим получить]

Дай формулу для русифицированной версии Excel.
Объясни, как работает.

[ПРИМЕР] Пример в действии

Задача: рассчитать премию сотрудника

Промпт:
Ты – эксперт по Microsoft Excel (русская версия). 
ЗАДАЧА: Создай формулу для расчета доплаты за выслугу лет. 

В организации доплата рассчитывается так:
- за каждый год стажа свыше 5 лет начисляется 1% от оклада. 
Оклад находится в ячейке B2, стаж — в ячейке C2. 
Формула должна учитывать, что если стаж меньше или равен 5 годам, 
доплата равна 0. Если стаж 5 лет и более – доплата = 1% от оклада. 

УСЛОВИЯ: если стаж меньше или равен 5 годам, доплата равна 0. 
Если стаж больше 5 лет, доплата = 1% от оклада за каждый год 
сверх 5 лет.

РЕЗУЛЬТАТ: сумма премии в рублях

[ФОРМУЛА] Ответ ИИ:

=ЕСЛИ(D2<15; 0; ЕСЛИ(D2<=20; B2*C2/100*0,5*(D2/E2); B2*C2/100*(D2/E2)))

С объяснением:
Формула проверяет:
- меньше 15 дней? → 0 рублей
- от 15 до 20 дней? → 50% от премии с учетом отработанных дней
- больше 20 дней? → полная премия пропорционально дням


РАЗДЕЛ 3: 10 типов формул, которые можно получить от ИИ
-----------------------------------------------------------------------------

[ТЕРМИН] Тип 1: Расчетные формулы

[ПРАКТИЧЕСКИЙ КЕЙС] Для расчета отпускных:

Напиши формулу Excel:
Рассчитать отпускные:
- средняя зарплата за 12 месяцев в B2
- количество дней отпуска в C2
- коэффициент 29,3 (среднее число дней в месяце)
Формула: средняя зарплата / 29,3 * дни отпуска

[ФОРМУЛА] Результат:
=B2/29,3*C2

[ТЕРМИН] Тип 2: Условные формулы (ЕСЛИ)

[ПРАКТИЧЕСКИЙ КЕЙС] Для табеля учета времени:

Напиши формулу:
Подсчитать тип дня в табеле:
- в ячейке A2 код: Я (явка), О (отпуск), Б (больничный), К (командировка)
- если Я – вернуть 8 (часов)
- если О или Б – вернуть 0
- если К – вернуть 8
- если пусто – вернуть "НЕ заполнено"

[ФОРМУЛА] Результат:
=ЕСЛИ(A2="Я"; 8; ЕСЛИ(ИЛИ(A2="О"; A2="Б"); 0; ЕСЛИ(A2="К"; 8; "Не заполнено")))

[ТЕРМИН] Тип 3: Поисковые формулы (ВПР/XLOOKUP)

[ПРАКТИЧЕСКИЙ КЕЙС] Для справочника должностей:

Напиши формулу поиска оклада по должности:
- в A2 написана должность сотрудника
- справочник должностей в диапазоне Лист2!A:B (колонка A - должности, колонка B - оклады)
- найти оклад для должности из A2
- если должность не найдена, показать "Нет в справочнике"

[ФОРМУЛА] Результат для современного Excel:
=ЕСЛИОШИБКА(ВПР(A2;Лист2!A:B;2;0); "Нет в справочнике")

[ТЕРМИН] Тип 4: Формулы подсчёта (СЧЁТЕСЛИ/СУММЕСЛИ)

[ПРАКТИЧЕСКИЙ КЕЙС] Для подсчёта больничных:

Напиши формулу:
Посчитать количество больничных дней сотрудника:
- диапазон с отметками: B2:AF2 (дни месяца)
- код больничного: "Б"
- подсчитать, сколько раз встречается "Б"

[ФОРМУЛА] Результат:
=СЧЁТЕСЛИ(B2:AF2;"Б")

[ТЕРМИН] Тип 5: Формулы дат

[ПРАКТИЧЕСКИЙ КЕЙС] Для расчета стажа:

Напиши формулу расчёта стажа работы:
- дата приёма в B2
- текущая дата - функция СЕГОДНЯ()
- результат в полных годах

[ФОРМУЛА] Результат:
=РАЗНДАТ(B2;СЕГОДНЯ();"Y")

[ТЕРМИН] Тип 6: Текстовые формулы (СЦЕПИТЬ, ЛЕВСИМВ, ПРОПИСН)

[ПРАКТИЧЕСКИЙ КЕЙС] Для автоматического формирования реквизитов документов:

Напиши формулу Excel: сформировать номер входящего документа по шаблону:
- год из ячейки A2 (2024)
- номер дела из ячейки B2 (015)
- порядковый номер из ячейки C2 (127)
Формат: "01-{год}/{номер дела}-{порядковый номер}" 
Пример: "01-2024/015-127"
Убери лишние нули у номеров.

[ФОРМУЛА] Результат:
="01-" & A2 & "/" & ТЕКСТ(B2; "000") & "-" & ТЕКСТ(C2; "000")

[ТЕРМИН] Тип 7: Проверочные формулы (ЕСЛИОШИБКА, ЕНД)

[ПРАКТИЧЕСКИЙ КЕЙС] Для контроля корректности данных в отчетах:

Напиши формулу:
Проверить корректность данных в отчете:
- если ячейка B2 содержит число - оставить как есть
- если текст "н/д" или пусто - вернуть 0
- если ошибка (например, #ДЕЛ/0!) - вернуть "Проверить"

[ФОРМУЛА] Результат:
=ЕСЛИ(ИЛИ(B2="н/д"; ЕПУСТО(B2)); 0; ЕСЛИОШИБКА(B2; "Проверить"))

[ТЕРМИН] Тип 8: Формулы для работы с бюджетами (СУММЕСЛИМН)

[ПРАКТИЧЕСКИЙ КЕЙС] Для анализа исполнения сметы:

Напиши формулу:
Рассчитать сумму расходов по статье "Командировки" за 4 квартал 2024 года (с 01.10.2024 по 31.12.2024):
- диапазон сумм: C2:C1000
- статьи расхода: B2:B1000
- даты: A2:A1000

[ФОРМУЛА] Результат:
=СУММЕСЛИМН(C2:C1000; B2:B1000; "Командировки"; A2:A1000; ">=01.10.2024"; A2:A1000; "<=31.12.2024")

[ТЕРМИН] Тип 9: Формулы округления (ОКРУГЛ, ОКРУГВВЕРХ)

[ПРАКТИЧЕСКИЙ КЕЙС] Для бюджетных расчетов по нормативам:

Напиши формулу:
Рассчитать сумму финансирования по нормативу:
- число получателей в B2 (137)
- норматив на человека в C2 (1250,75 руб.)
- округлить до целых рублей в большую сторону

[ФОРМУЛА] Результат:
=ОКРУГВВЕРХ(B2*C2; 1)

[ТЕРМИН] Тип 10: Формулы для работы с категориями (ВПР + ПОИСКПОЗ)

[ПРАКТИЧЕСКИЙ КЕЙС] Для автоматической классификации расходов:

Напиши формулу:
Определить вид расхода по коду бюджетной классификации:
- код в ячейке A2 (000 000 00000)
- справочник кодов на листе "КБК" (A:B)
- вернуть название вида расходов из второй колонки
- если код не найден, вернуть "Не определено"

[ФОРМУЛА] Результат:
=ЕСЛИОШИБКА(ВПР(A2; КБК!A:B; 2; 0); "Не определено")

[ТЕРМИН] Техника "Массив данных"

Напиши формулу для массива:

Есть таблица командировок (A2:E100):

РАЗДЕЛ 7: Готовые промпты для типовых задач
-----------------------------------------------------------------------------

[ПРАКТИЧЕСКИЙ КЕЙС] Расчет премии с КПЭ

Напиши формулу расчета квартальной премии:
- оклад в B2
- КПЭ сотрудника (0-100%) в C2
- общий фонд премирования отдела в D2
- количество сотрудников в E2

Логика:
- если КПЭ < 50% – премии нет
- если КПЭ 50-80% – премия = оклад * 0.5 * (КПЭ/100)
- если КПЭ > 80% – премия = оклад * 1 * (КПЭ/100)
- но не более чем фонд / количество сотрудников

[ПРАКТИЧЕСКИЙ КЕЙС] Расчет командировочных

Формула для расчета командировочных:
- дней командировки в B2
- город в C2
- справочник суточных по городам в Лист2!A:B

Правила:
- Москва/СПб – 700 руб/день, 
- в других городах – 500 руб/день
- если больше 10 дней – применяется коэффициент 0.9

[ПРАКТИЧЕСКИЙ КЕЙС] Учёт рабочего времени

Формула подсчета отработанных часов за месяц:
Диапазон B2:AF2 – отметки по дням
- "Я" = 8 часов (явка)
- "К" = 8 часов (командировка)
- "С" = 4 часа (сокращённый день)
- остальное = 0

Итого часов за месяц


РАЗДЕЛ 8: Частые ошибки и как их избежать
-----------------------------------------------------------------------------

[ОШИБКА] Ошибка 1: забыли про русский Excel

[СРАВНЕНИЕ]
❌ Неправильно:
=IF(A1>10,B1,C1)

✅ Правильно:
=ЕСЛИ(A1>10;B1;C1)

Всегда указывайте: "Формула для русского Excel"

[ОШИБКА] Ошибка 2: не указали диапазоны

[СРАВНЕНИЕ]
❌ Плохо: Посчитай сумму зарплат

✅ Хорошо: Посчитай сумму зарплат в диапазоне B2:B50

[ОШИБКА] Ошибка 3: слишком сложно в одной формуле

Решение: Разбейте на вспомогательные столбцы

Вместо одной мега-формулы сделай:
- В столбце F: промежуточный расчёт
- В столбце G: применение условий
- В столбце H: финальный результат
